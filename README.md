# 🕋 Hajj & Umrah Hotel Price Analysis

## Overview
This project analyzes hotel prices in Makkah, Madinah, and Taif during Hajj and Umrah seasons using data science and machine learning techniques. The objective is to understand how religious seasonality, location, and hotel characteristics influence accommodation prices and to build predictive models for hotel price estimation.

## Project Objectives
- Analyze hotel price patterns across different cities.
- Measure the impact of Hajj and Umrah seasons on hotel prices.
- Explore the relationship between hotel ratings, reviews, and pricing.
- Compare hotel pricing between cities.
- Develop and evaluate machine learning models for hotel price prediction.

## Dataset
The project uses:
- Primary dataset collected from Google Hotels through SerpAPI.
- Secondary hotel booking dataset for comparison and validation.

The primary dataset includes:
- Hotel Name
- City
- Price (SAR)
- Rating
- Reviews
- Check-in & Check-out Dates
- Season Type
- Location Information

## Key Findings
- Hotel prices are highest in Makkah compared to Madinah and Taif.
- Prices increase significantly during Ramadan and the period before Hajj.
- Hotel ratings have only a weak correlation with price.
- Linear Regression achieved the best predictive performance among the evaluated models.
