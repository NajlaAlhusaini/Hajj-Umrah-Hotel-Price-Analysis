🕋 Hajj & Umrah Hotel Price Analysis
Project Overview

This project analyzes hotel prices in Makkah, Madinah, and Taif during Hajj and Umrah seasons using data science and machine learning techniques.

Objectives
Analyze hotel price trends.
Compare prices across cities.
Study seasonal demand.
Predict hotel prices using machine learning models.
Dataset
Google Hotels API (SerpAPI)
Secondary hotel booking dataset
Technologies
Python
Pandas
NumPy
Matplotlib
Scikit-learn
Google Colab
Machine Learning Models
Linear Regression
Decision Tree
Random Forest
Results
Linear Regression achieved the best performance.
Hotel prices were highest in Makkah.
Hotel ratings showed weak correlation with prices.
